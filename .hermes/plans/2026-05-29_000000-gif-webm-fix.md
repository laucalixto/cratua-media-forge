# Plano: Suporte consistente a GIF e WebM no build_command

**Data:** 2026-05-29
**Meta:** Fazer GIF e WebM funcionarem com comandos ffmpeg válidos.

---

## Diagnóstico

### GIF
`-c:v libx264` com output `.gif` → erro -22. O container GIF não aceita H.264.
GIF precisa de: encoder nativo (omitir `-c:v`), sem áudio, filtro de palette.

### WebM/VP9
Testar localmente o comando exato. Possíveis causas:
- `-pix_fmt` posicionado após `-vf` (incomum mas válido)
- Audio `libopus` + parâmetros `-ac`/`-ar` conflitantes

## Tarefas

### Task 1: Container-aware build_command — GIF sem codec e sem áudio

**File:** `crates/mediaforge-core/src/ffmpeg.rs`

Após `cmd.arg(input)`, adicionar early return para GIF:

```rust
// GIF: special handling — no video codec, palette filter, no audio
if params.container == crate::enums::Container::Gif {
    let mut vf = format!("fps={},scale={}:{}:flags={},split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse",
        match params.fps { crate::enums::FpsMode::Fixed(f) => f, _ => 10 },
        params.width, params.height, params.scale_algorithm.ffmpeg_flag());
    for f in &params.video_filters { vf.push_str(&format!(",{}", f.to_ffmpeg_string())); }
    cmd.arg("-vf").arg(vf);
    cmd.arg("-progress").arg("pipe:2").arg("-nostats");
    cmd.arg(output);
    return cmd;
}
```

### Task 2: Testes para GIF

**File:** `crates/mediaforge-core/src/ffmpeg.rs` (test module)

```rust
#[test]
fn gif_no_video_codec() {
    let p = EncodeParams { container: Container::Gif, fps: FpsMode::Fixed(10), width: 480, height: 270, ..Default::default() };
    let s = cmd_str(&p);
    assert!(!s.contains("-c:v"), "GIF should not have -c:v");
    assert!(s.contains("palettegen"), "GIF needs palette filter");
    assert!(!s.contains("-c:a"), "GIF should not have audio");
}

#[test]
fn gif_has_palette_filter() {
    let p = EncodeParams { container: Container::Gif, fps: FpsMode::Fixed(10), ..Default::default() };
    assert!(cmd_str(&p).contains("palettegen"));
    assert!(cmd_str(&p).contains("paletteuse"));
}
```

### Task 3: WebM — pular movflags e profile:v

**File:** `crates/mediaforge-core/src/ffmpeg.rs`

No match de preset, adicionar caso Copy (GIF scenario acima já cobre). Para WebM, só garantir que o preset não inclui coisas quebradas (já corrigido — Opus audio, movflags vazio).

Adicionar skip de `-profile:v` e `-movflags` para WebM:

```rust
// Profile (skip for VP9/AV1 — not supported)
if let Some(profile) = &params.profile {
    if !matches!(params.video_codec, VideoCodec::VP9 | VideoCodec::AV1 | VideoCodec::SVTAV1) {
        cmd.arg("-profile:v").arg(profile.ffmpeg_name());
    }
}
```
